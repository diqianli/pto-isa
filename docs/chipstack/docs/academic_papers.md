### 代表性论文重点解读

#### 1. UCNN: Exploiting Computational Reuse in Deep Neural Networks via Weight Repression

**会议**: ISCA 2018 (计算机体系结构顶级会议)

**核心创新**:
发现神经网络中大量权重值重复出现的现象，提出**权重压缩(Weight Repression)**技术，通过识别和复用重复权重来减少乘法运算次数，实现**2-4倍的计算加速**。

**技术要点**:
- 权重聚类与量化：将相似权重聚类为同一值
- 计算复用架构：设计专用硬件复用重复计算
- 内存访问优化：减少权重加载次数，降低能耗

**合作者**: Jiyong Yu, Rohit Agrawal, Mengjia Yan, Michael Pellauer (NVIDIA), Christopher Fletcher

**与ChipStack的关联**:

#### 2. Accelerating Sparse Data Orchestration via Dynamic Reflexive Tiling

**会议**: ASPLOS 2023 (体系结构与系统顶级会议)

**核心创新**:
针对稀疏数据处理的**动态自反分块(Dynamic Reflexive Tiling)**技术，能够根据数据稀疏模式动态调整计算分块策略，显著提升不规则数据访问模式的处理效率。

**技术要点**:
- 运行时数据流分析：实时分析数据访问模式
- 自适应分块策略：动态调整计算分块大小
- 多机构协作：UIUC、NVIDIA、MIT、UC Davis联合研究
